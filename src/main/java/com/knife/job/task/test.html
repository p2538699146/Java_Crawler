职位信息


岗位职责：

1.自动化测试；

2.开发测试用例；

3.编写测试框架。

岗位要求：

1.计算机相关专业本科及以上学历,熟悉软件工程；

2.具有3年以上测试开发、自动化测试经验；

2.熟练掌握Python、Java等开发语言；

3.熟悉Linux环境,熟悉mysql数据库；   

4.有测试工具开发或测试框架编写经验者优先；

5.有较强的主动性、责任心、抗压能力，热爱新技术学习和实践。


职能类别：硬件测试工程师软件工程师

公司信息

    宁波中茂网络科技有限公司成立于2002年，全国员工600余人，30余个直属分支机构，服务企业客户超过5万家，是中国最具规模的外贸B2B软件供应商与领先的网络营销软件服务商。公司专注外贸信息化服务，志在为外贸行业打造全链条、可持续的业务（管理及营销）解决方案与产品服务体系，同时致力于为外贸企业培养、输送人才，建立人才库，解决外贸行业人才需求。
    公司产品“富通天下”在2002年为国家科技部外贸软件行业***创新基金立项，2006年被评为“双软企业”，2016年在杭州成立云技术研发中心，2022年计划IPO上市并市值破千亿。
    2020年是富通天下新的战略发展阶段，公司拥有前瞻性的经营理念和人性化的管理理念，深谙人才才是企业根本，希望携手有志之士一起谱写新的篇章！
